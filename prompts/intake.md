# 基础信息录入脚本

## 开场白

```
我来帮你创建这位群友的 Skill。只需要回答 3 个问题，每个都可以跳过。
```

---

## 问题序列

### Q1：群友昵称/代号

```
这位群友怎么称呼？（昵称、花名或代号都行，多个字用 - 连接）

例：example_xiaoming
```

- 接受任意字符串
- 生成的 slug 统一用 `-` 连接（不用下划线）
- 中文自动转拼音再用 `-` 连接（"小明" → `example_xiaoming`，"老李" → `lao-li`）
- 英文直接小写 `-` 连接（"Big John" → `big-john`）

---

### Q2：基本信息

把年龄、性别、职业、兴趣爱好放在一个问题里，让用户一句话说完：

```
用一句话描述他的基本信息——年龄、性别、职业、兴趣爱好，想到什么写什么，跳过也行。

例：25岁 男 程序员 喜欢打游戏
```

从用户的回答中解析以下字段（缺失的留空）：
- **年龄**
- **性别**
- **职业**
- **兴趣爱好**

---

### Q3：性格画像

把 MBTI、星座、个性标签、主观印象全部合在一起，让用户自由描述：

```
用一句话描述他的性格——MBTI、星座、个性特点、你对他的印象，
想到什么写什么，跳过也行。

例：ENFP 双子座 话痨 爱开玩笑 经常发表情包
```

从用户的回答中识别并提取以下字段（缺失的留空）：
- **MBTI**：16 种标准类型
- **星座**：12 星座
- **个性标签**：从下方标签库匹配，也接受自定义描述
- **主观印象**：无法归类的自由描述，直接保留原文

#### 个性标签库

**聊天风格**：话痨 / 话少 / 爱发表情包 / 爱发语音 / 秒回 / 潜水党 / 刷屏党

**性格特点**：开朗 / 内向 / 幽默 / 严肃 / 急性子 / 慢性子 / 直率 / 含蓄 / 热情 / 冷漠

**兴趣爱好**：游戏迷 / 动漫迷 / 音乐爱好者 / 运动达人 / 美食家 / 旅行爱好者 / 科技迷

**沟通风格**：直接 / 绕弯子 / 毒舌 / 温柔 / 理性 / 感性 / 爱抬杠 / 爱附和

## 确认汇总

收集完毕后展示：

```
信息汇总：

  👤  {昵称}
  🎂  {年龄}岁（若未填则省略）
  ⚧   {性别}（若未填则省略）
  💼  {职业}（若未填则省略）
  🎯  兴趣爱好：{兴趣爱好}（若未填则省略）
  🧠  {MBTI} {星座}（若未填则省略）
  🏷️   个性：{标签列表}（若未填则省略）
  💬  印象：{印象文本}（若未填则省略）

确认无误？（确认 / 修改 [字段名]）
```

用户确认后进入 Step 2 文件导入。
